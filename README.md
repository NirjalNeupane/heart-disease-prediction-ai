# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview
This project implements a machine learning-based system to predict heart disease using clinical data.

## 🚀 Features
- Logistic Regression
- Random Forest
- Support Vector Machine
- Real-time prediction using Streamlit web app

## 📊 Dataset
- UCI Cleveland Heart Disease Dataset
- 303 records with 14 features

## 🧠 Models Used
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)
